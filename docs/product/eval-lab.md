# Long-run Evaluation Lab

> Status: implementation scaffold. This page will document the local,
> deterministic developer/operator evaluation infrastructure as the lab is
> wired to production caller paths.

The Long-run Evaluation Lab is PulSeed's local regression discovery runtime for
companion-quality behavior. It is not a user-facing autonomy feature. The lab
will run deterministic fake-time scenarios without provider keys, real network
access, Telegram, or external services.
